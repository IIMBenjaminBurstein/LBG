Projet réalisé pour l'entreprise La belle games dans le cadre d'un projet scolaire.

Ce projet consiste en la refonte de leurs site web à partir d'une maquette travaillé avec des étudiants de l'axe créa et design.

Réalisation du site avec kirby cms, les tâches ont été :
   
   -création d'un template
   -intégration de maquette
   -Récupération des informations de contact dans un google sheet
   -création d'une interface administrateur
